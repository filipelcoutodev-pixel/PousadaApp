PousadaAPP - Sistema para Gerenciamento de Pousada

Status: Em Desenvolvimento.
Tecnologias Utilizadas: MySQLWorkbench, Java.
Desenvolvido por Filipe Luiz Couto.
O Sistema sera usado para gerenciar a Pousada.
É em Sistema Desktop, que permite:
- Cadastro de Hospedes com a escolha do quarto;
- Exclusão de Hospedes;
- Cadastro de Gastos;
- Visualização de Caixa com entradas e saidas;
- Emissão de relatorios completos com dados de todos os hospedes, mais valores de entradas, saidas e saldo;
- Emissãao de Relatórios Resumidos, contendo apenas Entradas, saidas e saldo;
- Busca de Hospede para Visualização seus dados e quarto(Função Portaria);
- Função de Permissão de Usuarios, acesso reduzido para usuários com tipos diferentes de Proprietario, Proprietaria e Administrador. 
   
   Para Utilização do Sistema é necessário um computador com impressora, com  sistema Operacional Windows, um Sistema de Gerenciamento de Banco de Dados(SGBD), e o java JRE instalados.
   (Internet Opcional)
